El problema planteado ha sido resuelto para que pueda comprobarse por consola, sin acudir a ninguna interfaz gráfica para hacerlo.

De este modo, la clase kelisto.rb resuelve el ejercicio completo que se plantea.
Para ello:
	1- Descargue dicho archivo en su ordenador local.
	2- Abra cmd y poga la ruta donde tenga el archivo descargado.
	3- Ejecute "ruby kelisto.rb"

Ahora aparecerá a modo informativo los productos disponibles que se pueden añadir al carrito de compra.
Luego le pedirá la cantidad que desea añadir de cada uno de ellos. Se debe introducir un número de cantidad válido.
Si se introduce letras, no se añadirá dicho producto y si se introduce número decimal, se añadirá sólo lo que corresponda a la parte entera.
Lo correcto es añadir números enteros (cantidades).

Como resultado final, por consola se mostrará el ticket de compra con el resultado final del precio y los descuentos que proceden aplicados.
